Projects, codes, work i've done

Here you are are going to find a few projects within the scope of my studies.
I have done them with some of my fellow students.

Recursive Triangles

The output of our function draws a Sierpinsky Triangle, which essentially is a big triangle whose midpoints are connected to create a new triangle. Again the midpoints of the new inner triangle are connected to create another triangle - this is done recursively to a specific degree (which we set to 7). The result is a beautiful theoretically endless pattern, which represents the recursion method.

Getting from A to B

In this project we were assigned to design and implement a Weighted Graph, in order to solve the shortest path problem with Dijkstra's algorithm. Our small application creates a random weighted graph. The user determines the number of edges and nodes that the graph should consist of. Afterwards the application prints the shortest as well as the cheapest path to the console.

Eight Queens Problem

Did you ever wonder whether it is possible to put eight queens on a chessboard without them threatening each other? Yes it is! We worked together on a backtracking algorithm which computes all possible combinations of this Eight Queens Problem.

Scrabble Cheater

Are you a desperate scrabble player? Use a scrabble cheater! With the help of a hashtable the Scrabble Cheater Application reads in any given tile rack and prints all English words "buildable" with the given letters.
